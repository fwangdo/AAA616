# Interval Analyzer
**요약해석(Abstract Interpretation) 프레임워크**를 이용하여 
**Interval Domain 기반의 정적 분석기(Interval Analyzer)** 를 구현하였습니다.

---

## 🧩 프로젝트 개요

이 분석기는 간단한 **While 언어**를 대상으로, 각 변수의 가능한 값의 범위를 추론하는  
**인터벌(Interval) 해석 기반의 정적 분석기**입니다.  
프로그램을 **제어 흐름 그래프(Control Flow Graph)** 로 변환한 뒤,  
워크리스트 기반의 **고정점(Fixpoint) 반복 알고리즘**을 통해 각 노드에서의 추상 메모리를 계산합니다.

---

## 🧠 주요 구성 요소

| 구성 요소 | 설명 |
|------------|------|
| **AST 정의** | While 언어의 산술식(`aexp`), 논리식(`bexp`), 명령문(`cmd`) 구조 정의 |
| **CFG 생성기 (`cmd2cfg`)** | 프로그램을 제어 흐름 그래프로 변환 |
| **추상 불 값(`AbsBool`)** | 참/거짓/모름(Top)/불가능(Bot)을 표현하는 추상 불 도메인 |
| **인터벌 도메인(`Interval`)** | 변수의 값 범위를 `[lower, upper]` 형태로 추상화 |
| **추상 메모리(`AbsMem`)** | 변수 이름을 인터벌 값에 매핑하는 추상 메모리 구조 |
| **고정점 계산(`analyze`)** | 워크리스트 기반으로 CFG를 순회하며, 각 노드의 추상 상태를 수렴시킴 |

---

## ⚙️ 분석 과정 요약

1. **While 프로그램 파싱 및 CFG 생성**  
   - 각 명령문(`Assign`, `If`, `While`, `Seq`)을 노드 단위로 변환  
   - 노드 간 제어 흐름 관계(edge) 생성  

2. **초기 추상 메모리 설정**  
   - 모든 변수는 `⊥`(bottom, 정의되지 않음) 상태로 시작  

3. **워크리스트 기반 고정점 반복**  
   - 각 노드에 대해 추상 상태를 전파  
   - `join`, `widen`, `narrow` 연산을 통해 수렴 유도  

4. **최종 추상 상태 테이블 출력**  
   - 각 프로그램 지점(Node)에서의 변수 범위를 출력
